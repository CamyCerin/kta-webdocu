## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

## Built with 
* **Front-end**
  * Vue.js
    * Vue router
    * Vue Page Transition
    * Vuejs VR
    * Vue2 touch eventsource
  * Axios
* **Back-end**
  * PHP
  * MySql
* **Cloud-Provider**
  * AWS
